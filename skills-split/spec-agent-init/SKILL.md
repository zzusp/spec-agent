# spec-agent-init

Use this skill only for state initialization.

- If the file defines naming conventions or mandatory constraints, apply them when generating `name/title`.

- `raw_requirement` (required)
- `name` (generated by caller AI)

```bash
python scripts/spec_agent.py init --name <name> --title "<title>" --desc "<raw_requirement>" --state-only
```

## Output

- Set `spec/.active`.
- Create clarification baseline files only.
